# TravelBuddy  - A Curated Adventure Marketplace
Project done using python , The objective of this project is to build a platform where companies can scale their business while the users can actively utlize the services.

**Travel Buddy** is a comprehensive web application designed to solve the fragmented and opaque nature of modern travel booking. It serves as a centralized marketplace where users can discover, compare, and book verified trips tailored to their adventurous needs.

---

## 🌟 Overview

Many travel booking methods today lack transparency, requiring users to manually verify trip organizers and navigate inconsistent itineraries. **Travel Buddy** provides a secure and clear path for experiential travel by acting as both a booking tool and a promotional hub for travel enthusiasts, hotels, and agencies.

A core pillar of this project is the promotion of **group traveling**. This fosters a social environment to meet like-minded people, expand connections, and reduce individual expenses while allowing solo travelers to experience a vibrant "group vibe" in a safe and friendly setting.

## ✨ Key Features

- **Verified MarketPlace**: Discover and book trips from verified organizers and agencies.
- **Group Travel Hub**: Join group trips to reduce costs and meet like-minded adventurers.
- **Search & Filter**: Find experiences based on destination, price, category, and organizer reputation.
- **Organizer Profiles**: A dedicated space for agencies and hotels to showcase unique experiences and build trust.
- **Social Integration**: Build a foundation of trust within the traveler community through reviews and reputation systems.

## 🛠️ Tech Stack

- **Front-End**: HTML5, CSS3, JavaScript (Vanilla)
- **Back-End**: Django (Python Framework)
- **Database**: SQLite (Development)
- **Email System**: OTP-based verification and automated booking notifications.

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- Django 4.x

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abhith1199/TravelBuddy.git
   cd TravelBuddy
   ```

2. **Create and activate a virtual environment:**
   ```bash
   python -m venv env
   # Windows
   .\env\Scripts\activate
   ```

3. **Install dependencies:**
   ```bash
   pip install -r travel_buddy/requirements.txt
   ```

4. **Navigate to project directory:**
   ```bash
   cd travel_buddy
   ```

5. **Apply migrations:**
   ```bash
   python manage.py migrate
   ```

6. **Run the development server:**
   ```bash
   python manage.py runserver
   ```

7. **Access the application:**
   Open `http://127.0.0.1:8000/` in your browser.
